# C#

<h1> C# - MiniBank Application </h1>


<h2>Description</h2>
Project consists of basic banking operations through a simple text-based menu. This project was built incrementally using Git and GitHub. Each feature was committed separately to demonstrate the development process.<br />

<h2>Features Added</h2>

- <b>Initial menu and balance<b>
- <b>Deposit feature<b>
- <b>Withdraw money<b>
- <b>Insufficient funds validation<b>
- <b>Invalid options<b>
<br />


<h2>Languages and Utilities Used</h2>

- <b>CFG Kickstarter Class</b>
- <b>C#</b>
- <b>.NET</b>
- <b>VS Code </b>
